## 📝 测试步骤

### 第一步：后端服务测试

1. **启动后端服务**
   ```bash
   cd backend
   npm start
   ```

2. **运行测试脚本**
   ```bash
   node test-new-features.js
   ```

3. **检查测试结果**
   - 权限管理 API
   - 员工管理 API
   - 绩效汇总 API

### 第二步：管理后台测试

1. **启动管理后台**
   ```bash
   cd admin
   npm run dev
   ```

2. **访问页面**
   - 权限管理：`http://localhost:5173/#/permissions`
   - 员工管理：`http://localhost:5173/#/employees`
   - 绩效汇总：`http://localhost:5173/#/performance-summary`

3. **功能测试**
   - 新增/编辑/删除操作
   - 数据筛选
   - 导出功能
