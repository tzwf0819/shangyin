## ⚠️ 已知问题

### 问题 1：后端路由未注册

**描述**: `app.js` 中未注册权限管理路由

**影响**: 权限管理 API 无法访问

**解决方案**: 
```javascript
// 在 app.js 中添加
app.use('/shangyin/permissions', require('./routes/permissions'));
```

**状态**: 🟡 已创建路由文件，待更新 app.js

### 问题 2：员工管理 API 未更新

**描述**: `employeeController.js` 未完全支持新字段

**影响**: 创建/更新员工时新字段可能不生效

**解决方案**: 更新 validateEmployeeData 和 create/update 函数

**状态**: 🟡 部分完成

### 问题 3：企业微信未配置

**描述**: 需要申请企业微信并配置参数

**影响**: 通知功能无法测试

**解决方案**: 
1. 申请企业微信
2. 获取 CorpID、AgentId、Secret
3. 配置到 `wecom_configs` 表

**状态**: ⏸️ 暂停
